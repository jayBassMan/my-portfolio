Workday Scheduler

A simple calendar application that allows a user to save events for each hour of the day.

using the Moment.js library to work with date and time.

Use a daily planner to create a schedule User is presented with time-blocks for standard business hours Each time-block will be color coded to indicate whether it is in the past, present, or future

Technologies:
  • - JavaScript
  • - Momentjs
  • - HTML
  • - CSS
